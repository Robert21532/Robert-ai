Robert AI — Gesture Intelligence Platform
This is a modern interactive web application that uses hand gesture recognition to create visual effects and interactive experiences. Here's what it does:

Core Technology
MediaPipe Hands API: Real-time hand tracking via your device's camera
Canvas Rendering: Multiple layered canvases for visual effects
Web Audio API: Sound effects and interactive audio
Three Interactive Modes


Play Mode (Default)

Pinch your thumb and index finger to trigger lightning effects ("zap")
Using both hands creates mandala patterns and lightning
Fast hand movements trigger animated rain effects
Real-time visual feedback with particles and ripples

Shapes Mode

Bring both palms together near your face to create a heart shape
Make a peace sign (index and middle fingers up) for specific gesture
Make fists with both hands to create star explosions
Write Mode

Use your index finger to draw in the air on screen
Pinch to lift the pen (pause drawing)
Shake your palm to clear the canvas
Includes a visual progress ring for shake gestures

Visual Design
Dark cyberpunk aesthetic with black background
Neon accent colors: Teal, pink, and blue gradients
Glass-morphism UI: Frosted glass effect panels with blur
Animated elements: Pulsing rings, shimmer effects, grain overlay
Real-time stats: FPS counter and hand count display
Toast notifications: Large animated text feedback for gestures

Features
Full-screen immersive experience
Responsive design (works on mobile with camera)
Mode-switching controls in the top bar
Help panel explaining gestures for current mode
Gesture detection with customizable confidence thresholds
Audio feedback (zap sounds, pop effects)


The experience is designed to be interactive, visually stunning, and engaging — perfect for demos, art installations, or entertainment purposes.
